### 4.4 정성적 비교

#### LSI의 강점
1. **수학적 정확도**: SVD의 최적 저차원 근사
2. **설명 분산**: 0.0935로 분산 설명 가능
3. **계산 효율**: 단일 SVD 분해로 빠른 학습
4. **음수 가중치**: 반대 개념 표현 (예: 긍정 vs 부정)

#### NMF의 강점
1. **해석 가능성**: 비음수 제약으로 명확한 부분 기반 표현
2. **희소성**: L1 정규화로 46.61% 희소성 달성
3. **직관적 가중치**: 모든 값이 양수로 기여도 명확
4. **부분 기반 표현**: 토픽 = 키워드의 가산 조합

#### 차이점 요약
- **LSI**: 전체 구조 파악, 잠재 의미 공간
- **NMF**: 부분 기반 표현, 해석 용이성
- **일치도**: NMI 0.4858 vs 0.3735 (차이 있음)

1. **비음수 제약의 효과**

   - 토픽 = 키워드의 가산 조합
   - LSI의 음수 가중치 문제 해결

   - 문서-토픽 분포 희소성: 46.61%

   - LSI(0.3735)와 큰 차이
   - 중간 일치도로 상호 보완적

4. **토픽 키워드 중복도**
   - NMF-LSI 평균 중복: 4.0개 (상위 10개 중)

5. **최대 토픽: 지능형 의사결정 시스템**

   - 대표 키워드: multi, intelligence, artificial, artificial intelligence, learning

### 8.3 최종 권장사항

**최종 선택**: **NMF**

NMF를 최종 선택합니다.

**선택 근거**:
1. **해석 가능성**: 비음수 제약으로 명확한 키워드 해석
2. **비즈니스 활용**: 토픽의 의미가 직관적이어서 실무 적용 용이
3. **희소성**: 46.61%로 의미있는 값에 집중
4. **K-means 일치도**: NMI 0.4858로 LSI 대비 우수
5. **Phase 5-6 활용**: 동향 분석 및 인사이트 도출에 명확한 토픽 필요

**LSI 대비 장점**:
- 음수 가중치 없음 → 키워드 기여도 명확
- L1 정규화 → 희소한 표현으로 핵심 키워드 강조
- 부분 기반 표현 → "이 토픽 = 이 키워드들의 조합" 직관적 설명 가능

**활용 방안**:
- Phase 5 (동향 분석): NMF 토픽으로 시간적 변화 추적
- Phase 6 (가속도 분석): NMF 토픽의 성장/하락 속도 측정
- 최종 리포트: NMF 토픽 기반 비즈니스 인사이트 도출
